Zip Solver

How to Use Solver:

  1. Adjust the grid size using the slider (5x5 to 10x10).
  2. Click on cells in the grid to select points in a specific order. These points will be numbered.
  3. Right-click near the edge of a cell to place a block that prevents the path from crossing that edge. Right-click the block again to remove it.
  4. Click "Solve" to find the path. The solution will be highlighted in green with numbered steps.
  5. Click "Clear" to remove all selected points, blocks, and any solution path from the grid.
  6. If the puzzle is too complex or has no solution, the solver may time out or alert that no solution was found.

Queens Solver

How to Use Solver:

Instructions:

  1. Define Regions: Click and drag on the grid to draw colored regions. Each distinct area you draw will be a new region. You can draw over existing regions to reassign cells. A single click on an unassigned cell starts a new region; dragging through an existing region's cell allows you to expand it.
  2. Place Queens (Optional): Right-click on a colored cell to place a queen. Right-click again to remove it.
  3. Solve: Click "Solve".
  4. Note: The grid must have N regions for an N x N grid, and all cells must belong to a region before solving.
